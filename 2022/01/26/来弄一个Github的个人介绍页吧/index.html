来弄一个Github的个人介绍页吧~

分类： 奇淫技巧
标签： Github

日期：2022-01-26 22:54:15

现在GitHub可以添加个人介绍页了，这里小小的分享一下吧~
先来看看效果是啥样：

是不是感觉效果还不错？实现起来还是非常简单的，开搞！

新建仓库

首先，需要新建一个和自己名字一样的仓库，名字一样的话会有一个小提示哦 o(￣︶￣)o

自我介绍

建好仓库后，就把仓库Clone下来，然后新建一个README.md文件，在里面写自我介绍；
其实，Github就是把你同名的仓库的README.md文件表现现实在首页而已，很好理解的。

自我介绍咋个写我就不多说了，这里说两个东西：

  • Github Stats Card：就是我截图里面的一个统计卡片；
  • Emoji： 就是一些可以放在Markdown文件的表情。

Emoji

这个玩意儿也非常的实用，可以让你很轻松的嵌入表情，进入Emoji官网选择喜欢的表情代码，复制粘贴即可。
